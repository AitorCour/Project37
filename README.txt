## About the game:
Dead Leaves, an old school survival horror

Controls

* W / S - Player Movement
* A / D - Player Rotation
* I - Inventory
* Space - Shoot mode
* O - Shoot
* U - Interact
* Enter - Confirm selection
* Escape - Exit
